## 即將推出

完整的操作步驟 — 前置需求、確切的環境變數、驗證步驟、還有踩雷提醒 —
會在後續的 slice 補上。在那之前，要設的環境變數是
[drop-in 對照表](../#find-your-tool)的第 6 列：

```bash
export OPENAI_HOST=http://localhost:9000
```

來源：[Goose installation guide](https://block.github.io/goose/docs/getting-started/installation)。
