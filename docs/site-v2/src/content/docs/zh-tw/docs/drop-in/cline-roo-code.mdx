## 即將推出

完整的操作流程——前置需求、確切的 UI 步驟、驗證步驟跟一些雷區——會在後續的
slice 補上。在那之前，要設定的就是
[drop-in 對照表](../#find-your-tool)的第 4 列：在 BYOK 設定裡把供應商選成
**Custom OpenAI**，然後把 Base URL 設成
`http://localhost:9000/v1`。

來源：[Cline BYOK 指南](https://docs.cline.bot/getting-started/byok)。
