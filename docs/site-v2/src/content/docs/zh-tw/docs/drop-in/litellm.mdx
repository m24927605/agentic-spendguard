## 即將推出

完整流程 — 前置需求、確切的環境變數、驗證步驟還有各種雷 — 會在後續的
slice 補上。在那之前,你要設的環境變數就是
[drop-in landing 對照表](../#find-your-tool) 的第 1 列:

```bash
export OPENAI_API_BASE=http://localhost:9000/v1
```

來源:[LiteLLM proxy config settings](https://docs.litellm.ai/docs/proxy/config_settings)。
