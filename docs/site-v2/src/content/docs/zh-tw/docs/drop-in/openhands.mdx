## 即將推出

完整流程 —— 前置需求、確切的 UI 步驟、驗證步驟還有那些雷 —— 會在後續
slice 補上。在那之前，要設的就是
[drop-in 對照表](../#find-your-tool)的第 5 列：到 Settings → LLM，新增一個
自訂供應商，把端點設成 `http://localhost:9000/v1` 就好。

來源：[OpenHands custom LLM configs](https://docs.all-hands.dev/usage/llms/custom-llm-configs)。
