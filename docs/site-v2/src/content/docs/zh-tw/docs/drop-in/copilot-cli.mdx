## 即將推出

完整教學 — 前置需求、確切的環境變數、驗證步驟還有踩雷提醒 — 會在後續的
slice 補上。在那之前，要設的環境變數就是
[drop-in 落地頁對照表](../#find-your-tool)的第 8 列：

```bash
export COPILOT_PROVIDER_BASE_URL=http://localhost:9000/v1
```
