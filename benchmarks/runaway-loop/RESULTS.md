## Methodology

- Each runner is configured with the same budget and a hard ceiling of 100 attempted calls.
- All calls hit a shared mock OpenAI-compatible endpoint that returns deterministic usage tokens.
- The mock LLM logs every call (model, tokens, runner) to a shared volume; the analyzer multiplies `(tokens × published price)` to get the ground-truth dollar number.
- Each runner self-reports its enforcement view; we surface both numbers so any drift between what the library *thinks* it stopped at and what *actually* hit the wire is visible.
- Pricing source: published OpenAI / Anthropic list prices, snapshot embedded in `analyze/analyze.py`.
